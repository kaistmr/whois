# 📂 카이스트 MR 동아리 인명사전 (Digital Handbook)

동아리 선후배 간의 네트워킹을 위한 웹 기반 인명사전입니다.  
**GitHub Pages(Hosting) + Google Apps Script(Bridge) + Google Sheets(DB)** 조합으로 구축되어 별도의 유지비가 발생하지 않습니다.

## 🚀 바로가기
- **웹사이트 주소:** https://kaistmr.github.io/whois/

---

## 🛠️ 운영 원리 (3줄 요약)
1. **관리자**가 구글 스프레드시트(`Master` 시트)에 부원 정보를 입력합니다.
2. 스프레드시트의 정보가 자동으로 웹용 CSV로 변환됩니다. (`Web` 시트)
3. **웹사이트**는 이 CSV를 읽어서 화면에 예쁘게 보여줍니다.

즉, **웹사이트 코드를 몰라도 엑셀만 고치면 반영됩니다!**

---

## 📅 매년 해야 할 일  

### 1. 신입 기수 데이터 업데이트
1. **구글 폼** 등을 이용해 신입생 정보를 걷습니다. (이름, 기수, 관심사, 번호 등)
2. 관리자용 스프레드시트의 `Master` 시트에 내용을 붙여넣습니다.
3. **끝!** 웹사이트에 자동으로 반영됩니다. (약 5분 뒤 새로고침 해보세요)

### 2. 졸업생 정보 업데이트
- 졸업생들의 소속(대학/직장)이 바뀌었다면 엑셀에서 수정해주면 됩니다.

---

## ⚠️ 문제 해결 (Troubleshooting)

### Q. 스프레드 시트의 링크를 바꾸고 싶어요.
  3. 만약 스프레드 시트의 링크가 바뀌었다면, MR 구글 드라이브의 Google App Scripts 파일 "Login_auth_server"의 설정에서 SECRET_SHEET_ID를 수정한 뒤, 해당 GAS를 새로 배포하여 배포URL을 깃허브의 config.js에서 바꿔주세요.

### Q. '설문조사' 버튼 링크를 바꾸고 싶어요.
- 깃허브 저장소에서 `config.js` 파일을 엽니다.
- `const SURVEY_URL = "..."` 부분의 따옴표 안 주소를 새 구글 폼 주소로 바꾸면 됩니다.

---

## 💻 개발자용 정보 (코드 수정 시)
- **`index.html`**: 웹사이트의 뼈대와 디자인이 들어있습니다.
- **`config.js`**: GAS의 배포 링크와 설문조사 링크 등 설정값이 들어있습니다.
- **'Login_auth_server'**: 로그인 인증 서버로서, 설정에서 구글 시트의 링크를 변경해줄 수 있습니다. 
- **기술 스택**: HTML, Tailwind CSS (CDN), Vanilla JS (프레임워크 X), GAS 기반 Login Auth Proxy
